# Butlerov desktop application

Electron-based desktop app built on the Butlerov editor and Vue 3. It packages the same drawing experience as the web component for a native windowed workflow.

**Status: under active development.** APIs, packaging, and platform support may change between releases. See the monorepo [README.md](../../README.md) for features and the [Vue component README](../vue/README.md) for embedding options in custom apps.

---

## Run the dev app

From the **repository root**:

```bash
npm i
npm run dev
```

This starts the app workspace in development mode (see `packages/app` scripts for details).
